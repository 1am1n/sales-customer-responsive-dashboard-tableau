# Sales & Customer Dashboard | Tableau Project

## Project Overview
This project presents an end-to-end interactive Tableau dashboard solution designed to analyze business sales performance and customer behavior.

The dashboards help stakeholders monitor KPIs, identify sales trends, evaluate profitability, and understand customer segments for better business decision-making.

---

## Key Features
- Sales performance tracking
- Revenue and profit analysis
- Customer segmentation analysis
- KPI monitoring dashboard
- Regional sales comparison
- Monthly and yearly trend analysis
- Interactive filters and drill-down reports
